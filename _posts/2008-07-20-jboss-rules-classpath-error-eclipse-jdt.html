--- layout: post title: JBoss Drools CLASSPATH Error; The Eclipse JDT Core jar is not in the classpath date: '2008-07-20T21:05:00.003-04:00' author: David Medinets categories: java modified_time: '2008-07-20T21:49:42.517-04:00' blogger_id: tag:blogger.com,1999:blog-3207985.post-4116180010340232036 blogger_orig_url: http://affy.blogspot.com/2008/07/jboss-rules-classpath-error-eclipse-jdt.html --- JBoss Rules v4.0.7 is incompatible with Eclipse v3.3 or higher. However there is a simple workaround. Run the following commands: 
export ECLIPSE_PLUGINS=/home/medined/Desktop/eclipse-3.4/plugins
ln -s 
  $ECLIPSE_PLUGINS/org.eclipse.jdt.core_3.4.0.v_874.jar 
  $ECLIPSE_PLUGINS/org.drools.eclipse_4.0.7/lib
