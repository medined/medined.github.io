--- layout: post title: Workaround For Minor Hibernate Bug - Blank hibernate.default_schema Property Adds Period date: '2008-04-18T09:42:00.003-04:00' author: David Medinets categories: java modified_time: '2008-04-18T09:44:37.505-04:00' blogger_id: tag:blogger.com,1999:blog-3207985.post-523121517726794163 blogger_orig_url: http://affy.blogspot.com/2008/04/workaround-for-minor-hibernate-bug.html year: 2008 theme: java --- We are using the hibernate.default_schema property with Oracle in production. Today I tried to use HSQL for our integration tests. When I blanked the hibernate.default_schema property, Hibernate tried to create tables named like '.FOO' - notice that pesky leading period! You can workaround this issue for HSQL by setting hibernate.default_schema to 'sa'.
